# 🎮 GameVerse Pro

GameVerse Pro is a modern multiplayer gaming platform built with Node.js, Express, Socket.io, MongoDB, and vanilla JavaScript.

It includes multiple games, user authentication, leaderboards, multiplayer support, and an admin dashboard.

---

# ✨ Features

- 🎮 Multiple Games
  - Ludo
  - Chess
  - Puzzle
  - Endless Runner
  - Survival

- 👤 User Authentication
- 🔐 JWT Login
- 🏆 Global Leaderboards
- 👥 Multiplayer via Socket.io
- ⭐ XP & Level System
- 🪙 Coins & Rewards
- 🛡 Anti-Cheat Protection
- 👨‍💼 Admin Dashboard
- 📱 Responsive UI

# 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Open project

```bash
cd GameVerse-Pro
```

Install dependencies

```bash
npm install
```

Copy environment file

```bash
cp .env.example .env
```

Run development server

```bash
npm run dev
```

Run production server

```bash
npm start
```
